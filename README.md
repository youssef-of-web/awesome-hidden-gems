## 🩺 Developer Life-Savers & Diagnostics

> Packages that diagnose problems, kill dead code, and save you hours. Run them with `npx` — no install needed.

| Tool | What it saves you from | Quick start |
|------|------------------------|-------------|
| **React Doctor** | Static-analysis audit for React — flags state, effects, performance, security & a11y issues (Next.js, Vite, RN). | `npx react-doctor@latest` |
| **react-scan** | Find & fix slow React re-renders — highlights them right in the browser. | `npx react-scan@latest localhost:3000` |
| **Knip** | Hunt down unused files, exports & dependencies in JS/TS projects. | `npx knip` |
| **depcheck** | Detect unused and missing npm dependencies. | `npx depcheck` |
| **npm-check-updates** | Upgrade your `package.json` deps to the latest versions in one shot. | `npx npm-check-updates -u` |
| **Madge** | Visualize module dependencies & catch circular imports. | `npx madge --circular src/` |
| **npkill** | Interactively find & delete heavy `node_modules` folders to free disk. | `npx npkill` |
| **size-limit** | Stop JS bundle bloat before it ships — fails CI on size regressions. | `npx size-limit` |
| **why-did-you-render** | Logs *why* a React component re-rendered — debugging gold. | `npm i @welldone-software/why-did-you-render` |
